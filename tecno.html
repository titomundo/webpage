Tecnología y software que uso

Computadoras

Tengo el privilegio de poder usar computadoras distintas para distintas tareas e intereses. Uso una computadora de escritorio, una laptop y una macbook prestada; y uso básicamente todos los sistemas operativos (Linux, MacOS y Windows).

Escritorio (Pc Custom) - DualBoot con Windows11 y Ubuntu

Computadora que armé junto con mi papá en 2021. Últimamente solo ve uso para jugar videojuegos y clases en línea ya que su SO primario es Windows 11. Recientemiente instalé un SSD con ubuntu, pero no he podido usarla mucho para programar ya que la mayoría de código que escribo es en mis laptops.

  • CPU: i5-9400F
  • GPU: GTX 1650
  • RAM: 32GB DDR4

Laptop Dell - Ubuntu

Esta es la laptop que he estado usando en el trabajo para programar páginas web. Personalmente me gusta más programar en linux ya que es posible tener un entorno más personalizable, seguro, con mejores herramientas y que consume menos recursos.

Tiene 16GB de RAM y un procesador AMD con gráficos integrados, honestamente no tiene demasiada potente y la batería podría ser mejor pero tiene capacidad de sobra para el uso que le doy.

2020 MacBook Air - Macos Sonoma

Esta laptop es la que uso para la escuela ya que llevo una materia de desarrollo para iOS. Me gusta para progamar igual que mi otra laptop, y no hay mucha diferencia práctica entre linux y macos a como yo los uso.

Software

Desde empezar a usar linux he descubierto muchas herramientas útiles para programar y he aprendido a manejar más la terminal y mi entorno de trabajo gracias a ello.

Neovim

Un editor de texto que corre en la terminal, el pricipal atractivo es que permite manipular y moverse entre el texto mediante el teclado sin tener que usar un mouse, lo cual permite codificar mucho más rápido una vez te acostumbras a los atajos.

Es bastante básico al salir de la caja, pero es altamente customizable mediante plugins, y una vez te tomas la molestia de configurarlo a tu gusto (añadir cosas como LSPs y autocompletar) es una muy buena opción como editor para programar. Aunque cuando programo en Windows, simplemente uso VSCode.

Kitty

Hablando de terminales, la terminal que uso es "kitty". Honetamente siento que no hay mucha diferencia entre emuladores de terminales, uso kitty porque es rápido y fácil de customizar.
